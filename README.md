# Nina Mason - Personal Portfolio Website

Welcome to the codebase for my personal portfolio!  
This website showcases my software engineering projects, my technical interests, and a little bit of my personality.  

It's built using **Next.js 16**, **React**, **TypeScript**, and **Tailwind CSS**.

---

## Features (planned + current)
- Clean, modern, responsive design  
- Personalized color palette + custom theme  
- Dynamic Projects page  
- Reusable React components (Navbar, Footer, Cards)  
- Accessible, semantic UI  
- Future integration with GPS Distance App project  
- Cozy + creative aesthetic (plants, crochet, pixel vibes)

---

## Tech Stack
- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Deployment:** Vercel (planned)
- **Version Control:** Git & GitHub
